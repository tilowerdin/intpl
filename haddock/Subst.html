-> Subst 

compose to substitutions with each other. the returned substitution has the following elements: let a -> b be a variable exchange of the first substitution (s1) then the goal substitution contains: a -> apply s2 b, when s2 is the second substitution. the goal substitution also contains all c -> d of s2 for all c that are not present in s1 on the left side.
